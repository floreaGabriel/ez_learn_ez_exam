⚠️ Cele 3 capcane care apar (aproape) mereu

1. NULL nu se compară cu =. WHERE Blocata <> null întoarce mereu UNKNOWN → 0 rânduri. Se scrie obligatoriu IS NULL / IS NOT NULL.
2. BETWEEN pe datetime „pierde” ziua finală. Data BETWEEN '2021-10-01' AND '2021-10-31' ratează orele de pe 31 oct. de după 00:00:00 (fiindcă '2021-10-31' = miezul nopții). Robust: Data >= '2021-10-01' AND Data < '2021-11-01'.
3. NOT IN (subinterogare) se „strică” la NULL. Dacă subinterogarea poate întoarce un NULL, NOT IN dă rezultat vid. Variantă sigură: NOT EXISTS sau LEFT JOIN ... WHERE ... IS NULL.

Clauzele fundamentale, cu exemple

Semantica NULL (IS [NOT] NULL)

-- gresit: comparatia cu null da mereu UNKNOWN
SELECT Cod_Bare FROM Volume WHERE Pierdut <> null;   -- 0 randuri (capcana!)
-- corect:
SELECT Cod_Bare FROM Volume WHERE Pierdut IS NOT NULL;

LIKE — cele 4 forme de potrivire (SQL Server)

WHERE Nume LIKE 'Pop%'      -- incepe cu "Pop"
WHERE Nume LIKE '%escu'     -- se termina in "escu"
WHERE Nume LIKE '%ana%'     -- contine "ana" oriunde
WHERE Nume LIKE '____u'     -- exact 5 caractere, ultimul "u" (_ = un caracter)
WHERE Cod_Bare LIKE '[1-9]%'   -- prima cifra nenula (clasa de caractere)

INNER JOIN în lanț 1—N—N

SELECT Nume, Prenume, Cod_Bare
FROM Cititori
INNER JOIN Imprumuturi ON Cititori.Id_Cititor = Imprumuturi.Id_Cititor
INNER JOIN Volume      ON Imprumuturi.Id_Imprumut = Volume.Id_Imprumut
WHERE Volume.Pierdut IS NULL;
-- in SQL Server inlantuiesti direct mai multe JOIN-uri, fara paranteze.

GROUP BY / HAVING — rețeta canonică

SELECT Nume, Prenume, COUNT(*) AS Nr
FROM Clienti
INNER JOIN Comenzi_Service ON Clienti.Id_Client = Comenzi_Service.Id_Client
WHERE Comenzi_Service.Stare = 'F'          -- filtreaza RANDURI (inainte de grupare)
GROUP BY Comenzi_Service.Id_Client, Nume, Prenume
HAVING COUNT(*) >= 2                        -- filtreaza GRUPE (dupa agregare)
ORDER BY COUNT(*) DESC;

Subinterogări: orfani și „niciuna” (NOT EXISTS)

-- inregistrari "orfane" (robust, fara capcana NOT IN):
SELECT P.Cod_Piesa FROM Piese_Montate P
LEFT JOIN Comenzi_Service C ON P.Id_Comanda = C.Id_Comanda
WHERE C.Id_Comanda IS NULL;

-- cuantificatorul "niciuna" (corelat):
SELECT Nume, Prenume FROM Pasageri P
WHERE NOT EXISTS (SELECT 1 FROM Rezervari R
                  WHERE R.Id_Pasager = P.Id_Pasager AND R.Stare = 'X');
